# 06 — 视觉设计系统

> 颜色、字体、间距、阴影、圆角、过渡。确定整个应用的视觉语言。

---

## 整体风格

**参考**：Netflix 媒体库 + Kindle 阅读器 + 工具后台

- 深色为主——减少阅读时的视觉疲劳
- 高对比度——漫画封面是主角
- 克制——大面积留白，少用装饰
- 安静——动画短而轻，不抢注意力

---

## 颜色

### 基础色板

| Token | 色值 | 用途 |
|-------|------|------|
| `--bg` | `#0f0f0f` | 页面背景 |
| `--surface` | `#1a1a1a` | 卡片、面板 |
| `--border` | `#2a2a2a` | 分割线、边框 |
| `--text` | `#9ca3af` | 正文、辅助文字 |
| `--text-h` | `#e5e7eb` | 标题、重要文字 |
| `--accent` | `#6366f1` | 主色调（按钮、链接、高亮） |
| `--accent-hover` | `#818cf8` | Hover 态 |
| `--code-bg` | `#1f2937` | 代码块、次要容器 |

### 语义色

| Token | 色值 | 用途 |
|-------|------|------|
| `--success` | `#22c55e` | 成功、完成 |
| `--warning` | `#f59e0b` | 进行中、警告 |
| `--danger` | `#ef4444` | 失败、删除、错误 |

### 当前 CSS 变量（`style.css`）

项目中已定义 `--bg`, `--text`, `--text-h`, `--border`, `--code-bg`, `--accent`, `--social-bg`。设计阶段保持现有变量名，增加缺失的 `--surface`、`--success`、`--warning`、`--danger`、`--accent-hover`。

## 组件 Token

### 卡片（ComicCard）

- **背景**：`#1a1a1a`（`--surface`）
- **边框**：无
- **圆角**：8px
- **Hover**：`transform: translateY(-2px)` + 阴影加深（200ms）

### 按钮

| 类型 | 背景 | 文字色 |
|------|------|--------|
| Primary | `--accent` | `#fff` |
| Default | `--surface` | `--text-h` |
| Danger | 透明 | `--danger` |
| Link | 透明 | `--accent` |

### 输入框

- **背景**：`#1a1a1a`
- **边框**：`--border`
- **Focus**：边框变为 `--accent`
- **圆角**：4px
- **内边距**：8px 12px

### 进度条

- **轨道**：`--border`
- **填充**：`--accent`
- **高度**：8px（列表）/ 6px（卡片内）

### 标签（Tag）

- **背景**：`rgba(99,102,241,.15)`
- **文字**：`--accent`
- **圆角**：4px
- **内边距**：2px 8px

### 状态标签

| 状态 | 颜色 |
|------|------|
| 等待中 / 信息 | `--text` |
| 进行中 / 下载中 | `--warning` |
| 成功 / 完成 | `--success` |
| 失败 / 错误 | `--danger` |

## 图片处理

### 封面

- **圆角**：8px
- **宽高比**：3:4
- **加载态**：骨架屏（背景 `--code-bg`，脉冲动画）
- **错误态**：灰色占位 + 图标

### 阅读器图片

- **最大宽度**：100%
- **居中**
- **无圆角**
- **间距**：上下 8px
