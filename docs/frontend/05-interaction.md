# 05 — 交互设计

> 页面怎么操作。只讨论行为，不讨论颜色、字体、圆角。聚焦点击、键盘、滚动、Hover、状态切换。

---

## Library（漫画库）

### 搜索

| 操作 | 行为 |
|------|------|
| 输入文字 | 300ms 后自动搜索，不显示 loading，结果直接替换列表 |
| 清空 | 恢复全部漫画列表 |
| Enter | 立即搜索（不等 debounce） |

### 标签筛选

| 操作 | 行为 |
|------|------|
| 点击下拉 | 显示所有标签 |
| 选择标签 | 立即筛选 |
| 清除 | 恢复全部 |

### 排序

| 操作 | 行为 |
|------|------|
| 点击下拉 | 显示排序选项 |
| 切换 | 立即重新排序 |

### 漫画卡片

| 操作 | 行为 |
|------|------|
| 点击 | 跳转 Comic Detail |
| Hover | 卡片阴影加深 |

### 分页

| 操作 | 行为 |
|------|------|
| 点击页码 | 加载对应页 |
| 搜索/筛选 | 重置到第 1 页 |

### 空状态

当没有任何漫画时，显示引导提示："还没有漫画，去导入一本吧" + 按钮跳转 `/import`。

---

## Comic Detail（漫画详情）

### 操作按钮

| 按钮 | 行为 |
|------|------|
| 继续阅读 | 跳转 Reader（上次位置） |
| 从头开始 | 跳转 Reader（第 1 章第 1 页） |
| 生成LQ | 发送请求 → Toast 提示"已提交" → Task Center 可查看 |
| 删除漫画 | 弹出确认 → 确认后删除 → 返回 Library |

### Catalog 树

| 操作 | 行为 |
|------|------|
| 展开/折叠 | 点击目录名（如 "Vol.1"）切换展开状态 |
| 点击章节 | 跳转 Reader（该章节第 1 页） |
| 默认状态 | 全部展开 |

---

## Reader（阅读器）— 核心页面

### 图片区域

| 操作 | 行为 |
|------|------|
| 点击左侧 1/3 | 上一页 |
| 点击右侧 2/3 | 下一页 |
| 鼠标滚轮 | 滚动查看当前页（长图） |
| 图片加载中 | 显示骨架屏（aspect-ratio 保持比例） |
| 图片加载失败 | 显示"加载失败"占位 |

### 工具栏

| 元素 | 行为 |
|------|------|
| 返回按钮 | 返回 Comic Detail |
| 章节标题 | 显示当前章节名 |
| 页码 | 当前页 / 总页数 |
| 上一章按钮 | 跳转上一章第 1 页 |
| 下一章按钮 | 跳转下一章第 1 页 |
| 设置按钮 | 打开设置抽屉 |

### 键盘快捷键（待实现）

| 按键 | 行为 |
|------|------|
| ← / A | 上一页 |
| → / D | 下一页 |
| Space | 下一页 |
| ESC | 返回 Comic Detail |

### 设置抽屉

| 选项 | 行为 |
|------|------|
| HQ 模式（开关） | 切换 HQ/LQ 图片源 |

### 进度记录

| 行为 | 触发条件 |
|------|----------|
| 自动同步 | 每翻 5 页自动保存 |
| 离开时同步 | 页面卸载时保存当前页 |

### 章节结束

读到最后一页时，"下一章"按钮高亮提示，或自动弹出"下一章"选项。

## Import（导入）

| 操作 | 行为 |
|------|------|
| 选择来源 | 下拉切换 ZIP / DIRECTORY |
| 输入路径 | 文本框输入 |
| Enter | 提交 |
| 提交按钮 | 发送请求 → Toast "导入任务已创建" → 清空输入 → 提示"前往任务中心查看进度" |

---

## Task Center（任务中心）

| 操作 | 行为 |
|------|------|
| 页面加载 | 立即拉取最新任务列表 |
| 自动轮询 | 每 3 秒刷新（有进行中任务时） |
| 取消 | 进行中任务 → 点击取消 → Toast "已取消" |
| 重试 | 失败任务 → 点击重试 → Toast "已重试" |

### 状态颜色

| 状态 | 左侧边框颜色 |
|------|-------------|
| 进行中 | 蓝色（accent） |
| 失败 | 红色 |
| 完成 | 绿色 |

## 通用交互原则

1. **即时反馈**：列表搜索、标签筛选不显示 loading，结果直接替换
2. **Toast 提示**：操作成功/失败使用轻量 Toast，不弹 Dialog
3. **空状态引导**：列表为空时显示引导文字 + 操作按钮，不显示空白页
4. **危险操作确认**：删除需要 Popconfirm 二次确认
5. **自动轮询**：任务中心 3 秒刷新，无需手动刷新按钮
