# Issue 记录模板

新建 issue 时复制本模板，按以下格式填写。

---

## 编号

ISSUE-XXX

## 类型

- [ ] BUG
- [ ] TODO
- [ ] ENHANCEMENT
- [ ] QUESTION

## 模块

Import / Task / Library / Comic Detail / Reader / History / UI / API / Other

## 描述

简明扼要地描述问题或需求。

## 复现步骤

1. 进入 xxx 页面
2. 点击 xxx
3. 输入 xxx
4. 观察到 xxx

## 期望行为

描述应该发生什么。

## 实际行为

描述实际发生了什么。

## 截图 / 日志

粘贴相关截图或日志。

## 影响范围

- 是否阻塞 Beta v0.1 发布？
- 是否影响全链路？
- 是否仅在特定条件下出现？

## 建议修复方案

可选。

## 状态

- [ ] 待处理
- [ ] 处理中
- [ ] 已修复
- [ ] 暂不修复
